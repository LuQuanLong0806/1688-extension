# 🔥 1688图片抓取 - Chrome扩展 安装说明

## 安装步骤

### 第1步：打开扩展管理页

在Chrome浏览器地址栏输入：

```
chrome://extensions/
```

按回车打开。

### 第2步：开启开发者模式

在页面右上角，打开 **"开发者模式"** 开关。

### 第3步：加载扩展

1. 点击左上角 **"加载已解压的扩展程序"** 按钮
2. 选择 `1688-extension` 文件夹（就是本文件所在的文件夹）
3. 点击 "选择文件夹"

### 第4步：使用

1. 打开任意 **1688商品详情页**
2. 先**滚动到底部**让所有图片加载出来
3. 点击浏览器右上角的 **扩展图标**（橙色G图标）
4. 点击 **"🚀 开始抓取图片"** 按钮
5. 新标签页中会显示所有抓到的图片！

## 功能

- ✅ 全选/取消全选图片
- 📋 一键复制所有图片地址
- 💾 下载图片地址列表
- 🔍 点击图片预览大图

## 注意事项

- 仅在 1688.com 商品详情页有效
- 建议先滚动页面到底部再抓取，确保图片全部加载
- 如果弹出窗口被阻止，请点击地址栏右侧允许弹出窗口

## 文件说明

- `manifest.json` - 扩展配置
- `popup.html` / `popup.js` - 弹出窗口界面
- `content.js` - 核心抓取脚本（在页面中执行）
- `icon.svg` - 扩展图标
